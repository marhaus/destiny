Methods

GET

Gets a page list of Destiny items.

request query parameters
parameter value description

count

int (required)

One of:

  • 10
  • 25
  • 50
  • 100
  • 500 (default)

Number of rows to return.

page

int (required)

Default: 0

Page number to return, starting with 0.

name

string

Default:

Name of items to return (partial match, no case). Omit for all items.

order

string (required)

One of:

  • Name (default)
  • ItemType
  • Rarity
  • ItemTypeName
  • ItemStatHash
  • MinimumRequiredLevel
  • MaximumRequiredLevel

Item property used for sorting.

orderstathash

string

Default:

This value is used when the order parameter is set to ItemStatHash. The item stat for the provided hash value will be used in the sort.

direction

string (required)

One of:

  • Ascending
  • Descending

definitions

string

Default:

Indicates the item definitions should be returned with item hash results.

step

string

Default:

Hash ID of the talent node step that an item must have in order to be returned.

categories

string

Default:

Category identifiers. Only items that are in all of the passed-in categories will be returned.

rarity

string

One of:

  • Currency
  • Basic
  • Common
  • Rare
  • Superior
  • Exotic

Rarity of items to return. Omit for all items.

weaponPerformance

string

One of:

  • RateOfFire
  • Damage
  • Accuracy
  • Range
  • Zoom
  • Recoil
  • Ready
  • Reload
  • HairTrigger
  • AmmoAndMagazine
  • TrackingAndDetonation
  • ShotgunSpread
  • ChargeTime

Items must have node steps in one of these categories, omit for all items.

impactEffects

string

One of:

  • ArmorPiercing
  • Ricochet
  • Flinch
  • CollateralDamage
  • Disorient
  • HighlightTarget

Items must have node steps in one of these categories, omit for all items.

guardianAttributes

string

One of:

  • Stats
  • Shields
  • Health
  • Revive
  • AimUnderFire
  • Radar
  • Invisibility
  • Reputations

Items must have node steps in one of these categories, omit for all items.

lightAbilities

string

One of:

  • Grenades
  • Melee
  • MovementModes
  • Orbs
  • SuperEnergy
  • SuperMods

Items must have node steps in one of these categories, omit for all items.

damageTypes

string

One of:

  • Kinetic
  • Arc
  • Solar
  • Void

Items must have node steps in one of these categories, omit for all items.

sourcecat

string

One of:

  • Vendor
  • Activity

Items must drop from the specified source category, omit for all items.

sourcehash

string

Default:

Items must drop from the specified source, omit for all items. Overrides sourcecat.

matchrandomsteps

string

Default:

True if the supplied groups/step hash filters should match random node steps. False indicates the item can always get the step before it is considered a match.

  • Status Code200-application/json; charset=utf-8(des:Response)
